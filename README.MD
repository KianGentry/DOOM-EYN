# DOOM FOR EYN-OS

It's here, at long last!

## Compilation

It compiles for and runs on EYN-OS, running on an i386 machine. I've only tested with 9MB+ of RAM, I can't say for sure how it performs on anything lower. It's acceptable performance on the i386 VM that I test EYN-OS in.

If you wanna compile it, please copy it to the EYN-OS source root directory. There's a good little helper tool there for compilation, and some EYN-OS headers are required for compilation to EYN-OS (namely gui.h).

## Running / Playing

Obviously, a valid WAD is required. I've only tested with DOOM Shareware 1.9. Any version below that doesn't seem to work. P.S. It takes a LONG time to boot up, be patient. 

Only mouse controls work, I couldn't get EYN-OS' funky keyboard handling to work to a standard I was happy with, so I just left it as mouse only. It's a bit finnicky, but after you get a feel for it, it works fine.

---

Other than all of that, enjoy playing DOOM on EYN-OS!
